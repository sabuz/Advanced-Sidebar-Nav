== Description ==
The best way to display navigation menus on sidebar, no matter how many depth!

== Installation ==
After downloading and extracting the latest version of the plugin:

1. Install using the WordPress Plugin installer, or Extract the zip file and drop the contents in the wp-content/plugins/ directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Appearance > Widgets.
4. Drag `Advanced Sidebar Nav` into the widget area.
5. Enjoy!

If you encounter any bugs, or have comments or suggestions, please submit a ticket using support forum.
